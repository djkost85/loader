	"need_update": "Нужно ли обновлять? 1 да 0 нет"

К новому формату необходимо добавить поле для хранения кода арендатора, предпологается использовать для этого время в милисекундах когда совершалась аренда и добавить в это время паузу, чтоб не смешивались коды, время аренды будет равно времени обновления прокси(опция выставляется в ручную), контроль за обновление прокси адресов должен осуществлять класс proxy
Классы использующие proxy прибегают только к методу запроса по идентификатору присвоеному лично им классом proxy для получение их арендованных прокси адресов.
Так же к каждому прокси адресу нужно присвоить к какому сайту он относится. Имя сайта заданое в ключах массива $this->url_proxy_list
Для создания очереди необходимо разработать функцию доступа к файлу, если он занят, ждать пока освободится, когда освободиться зпблокировать изменить и освободить с последующей разблокировкой.
